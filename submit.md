# Submit a Nomination

All nominations are submitted via our GitHub repository. You will need a free GitHub account.

[**Submit via GitHub Issue Form &rarr;**](https://github.com/YOUR_USERNAME/YOUR_REPO/issues/new/choose) *(Update this link to your repo!)*

### Guidelines
* Please search existing nominations before submitting to avoid duplicates.
* Submissions are initially marked as `status: pending`.
* Maintainers will review your submission. If it meets the Code of Conduct, it will be labeled `status: approved` and will automatically appear on this website within a few minutes.
